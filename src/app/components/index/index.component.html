¡Bienvenido!

Este proyecto consiste en un portfolio web totalmente editable, que se almacena en una base de datos. Permite la creación de varios usuarios e implementa seguridad JWT.

Uso de la App

Se puede ingresar o crear un nuevo usuario, haciendo click en el botón Log In en la esquina superior derecha. Si no desea iniciar sesión, pero desea ver algun perfil, puede ingresar a cualquier perfil en el directorio "/nombreDeUsuario". Una vez logueado, puede editar su perfil, utilizando los botones disponibles en cada elemento editable. El sitio también cuenta con drag and drop para poder reacomodar la información cuando sea necesario.
